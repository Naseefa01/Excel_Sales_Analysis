# Excel_Sales_Analysis
End-to-End Sales Data Analysis in Excel with Dashboard, Pivot tables, Charts &amp; Business Insights.

## Project Overview
This project analyzes sales data using Microsoft Excel. 
The dashboard provides insights into revenue, profit, and regional performance using Pivot Tables and Charts.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning
- Basic Formulas

---

## Key Features
- Sales by Region Analysis
- Customer performance
- Product Performance
- Monthly Revenue Calculation
- Interactive Dashboard

---

## Insights
- South region generated highest revenue.
- Electronics category contributes major share of total sales.
- Seasonal trends observed in data.
- Top 5 Customers are recorded.
